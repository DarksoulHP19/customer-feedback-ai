# Customer-feedback-ai

An AI agent system to analyze customer reviews, extract key themes, and prioritize product issues using FastAPI and LLMs.

## download datasets

[dataset](https://www.kaggle.com/datasets/arhamrumi/amazon-product-reviews) for product reviews

after downloading, place the dataset in the data folder and unzip it. The expected file structure is:

```customer-feedback-ai/
├── data/
│   └── reviews.csv
├── main.py
```
